# Grafische Oberflaechen unter Linux

## Schneller visueller Überblick

| Oberfläche | Merkmal |
|---|---|
| GNOME | klar, modern |
| KDE Plasma | sehr anpassbar |
| Xfce | leicht, schnell, ressourcenschonend |
| Cinnamon | klassischeres Desktop-Erlebnis |

## Was ist eine grafische Oberfläche unter Linux?

Die grafische Oberfläche ist die Schicht, die das Arbeiten mit Maus, Fenstern, Icons und Menüs ermöglicht.

Für den Benutzer sichtbar ist zum Beispiel:

- Anmeldebildschirm
- Desktop
- Panel oder Taskleiste
- Anwendungsmenü
- Dateimanager
- Systemeinstellungen

## Was ist eine Desktop-Umgebung?

Nicht nur ein „schönes Aussehen", sondern in der Regel ein komplettes Paket:

- Fenstermanager
- Panel oder Dock
- Dateimanager
- Einstellungsmanager
- Systemanwendungen
- Widgets oder Erweiterungen

## 1. GNOME

Merkmale:

- moderner, klarer Ansatz
- übersichtliche Oberfläche
- bei vielen Distributionen Standard oder häufige Wahl

## 2. KDE Plasma

Merkmale:

- außerordentlich anpassbar
- viele Einstellungsmöglichkeiten
- modern und funktionsreich

## 3. Xfce

Merkmale:

- leicht und schnell
- geringerer Ressourcenbedarf
- klassischeres Desktop-Erlebnis

## 4. Cinnamon

Merkmale:

- klassischeres Desktop-Layout
- benutzerfreundlicher Ansatz
- für viele eine vertraute, Windows-ähnliche Logik

## Warum ist es wichtig, dass es unter Linux mehrere Oberflächen gibt?

Denn in der Linux-Welt ist Wahlfreiheit üblich.

Das bedeutet:

- dasselbe System kann mit einer anderen Oberfläche verwendet werden
- der Benutzer kann die Umgebung an seine Bedürfnisse anpassen

Zum Beispiel:

- Leistung
- Einfachheit
- Aussehen
- Anpassbarkeit

## Verwandte Begriffe

### Display-Server

Die technische Schicht hinter der grafischen Darstellung.

Gängige Namen:

- `X11`
- `Wayland`

Für die Prüfung reicht es zu wissen:

- diese gehören zum technischen Hintergrund der grafischen Funktionsweise

### Window-Manager

Verwaltet das Verhalten und die Positionierung der Fenster.

## Linux-GUI vs Windows / macOS

| Linux | Windows / macOS |
|---|---|
| mehrere Desktop-Umgebungen | in der Regel einheitlichere Werksoberfläche |
| starke Anpassbarkeit | im Allgemeinen gebundeneres Grunderlebnis |
| hängt auch von der Distribution ab | vom Hersteller enger festgelegt |

## Prüfungstaugliche Formulierung

> Unter Linux können verschiedene grafische Oberflächen, auch Desktop-Umgebungen genannt, verwendet werden.  
> Beispiele dafür sind GNOME, KDE Plasma, Xfce oder Cinnamon.  
> Diese stellen die Fensterverwaltung, die Panels, den Dateimanager und die grafischen Einstellungsmöglichkeiten bereit.  
> Ein Vorteil von Linux ist, dass der Benutzer zwischen verschiedenen grafischen Umgebungen wählen kann.

## Häufige Prüfungsfehler

- Glauben, dass es unter Linux nur ein Terminal gibt.
- Linux und GNOME als dasselbe betrachten.
- Nicht mindestens 2-3 Linux-Grafikoberflächen aufzählen können.
- Den Window-Manager mit der vollständigen Desktop-Umgebung verwechseln.
- Nicht wissen, dass Wayland und X11 zur grafischen Hintergrundschicht gehören.
