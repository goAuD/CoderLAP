## Mit jelent az, hogy piacvezető?

Általában azt jelenti, hogy egy adott platform vagy operációs rendszer:

- sok eszközön fut
- széles körben elterjedt
- nagy felhasználói bázissal rendelkezik
- erős szoftver- és támogatási ökoszisztémája van

Ez lehet:

- telepített példányszám
- aktív használat
- piaci részesedés
- vállalati elterjedtség

## 1. Windows

A Windows továbbra is a legelterjedtebb **desktop** operációs rendszer.

Erősségei:

- nagyon széles hardvertámogatás
- sok üzleti és hétköznapi szoftver
- erős vállalati jelenlét
- játékoknál és általános PC-használatban nagyon erős

## 2. macOS

A macOS az Apple asztali operációs rendszere.

Erősségei:

- Apple hardverekre optimalizált
- stabil, egységes környezet
- erős integráció más Apple eszközökkel
- népszerű kreatív és fejlesztői környezetekben is

## 3. Linux

A Linux desktop részesedése kisebb, de szakmailag nagyon fontos.

Erősségei:

- nyílt forráskód
- sokféle disztribúció
- rugalmas testreszabás
- erős fejlesztői, rendszergazdai és szerveres jelenlét

## 4. Android és iOS

Összes platformot nézve az **Android** és az **iOS** nagyon meghatározóak.

Fontos azonban:

- ezek főleg **mobil operációs rendszerek**
- a témakatalógus külön is foglalkozik mobil hardverrel és Android / iOS ismeretekkel

## 5. Szerveroldali elterjedtség

Szervereknél a Linuxnak kiemelkedő jelentősége van.

Gyakori példák:

- Ubuntu Server
- Red Hat Enterprise Linux
- Debian
- SUSE
- Oracle Linux

Emellett fontos a:

- Windows Server

## Miért fontos vizsgán?

Mert itt nem elég csak neveket mondani, hanem tudni kell:

- melyik milyen környezetben erős
- mi a különbség desktop és mobil piac között
- miért nem ugyanaz a „piacvezető” minden kategóriában

## Vizsgán jól használható megfogalmazás

> A piacvezető operációs rendszerek kategóriától függnek.  
> Desktop környezetben a Windows a legelterjedtebb, mellette fontos a macOS és a Linux.  
> Mobil területen az Android és az iOS dominál.  
> Szerveres környezetben a Linuxnak és a Windows Servernek van nagy jelentősége.  
> A pontos piaci helyzet idővel változhat, ezért mindig meg kell mondani, milyen platformról és milyen időpontról beszélünk.

## Gyakori vizsgahibák

- Desktop és mobil piac összekeverése.
- Azt mondani, hogy a Linux mindenütt vezető.
- Azt mondani, hogy a macOS ugyanúgy általános, mint a Windows hardveroldalon.
- Nem megadni, hogy a piaci helyzet időfüggő.
- Androidot desktop operációs rendszerként említeni.

## Gyors önellenőrzés

1. Melyik a legelterjedtebb desktop operációs rendszer?
2. Melyik két rendszer dominál mobilon?
3. Melyik operációs rendszer család nagyon fontos szervereken?
4. Miért kell dátumot vagy időszakot mondani a piaci részesedésnél?
5. A macOS milyen hardverekhez kötődik?

## Rövid válaszok az önellenőrzéshez

1. A **Windows**
2. Az **Android** és az **iOS**
3. Főleg a **Linux**, de a **Windows Server** is fontos
4. Mert a piaci arányok idővel változnak
5. Az **Apple** hardvereihez
