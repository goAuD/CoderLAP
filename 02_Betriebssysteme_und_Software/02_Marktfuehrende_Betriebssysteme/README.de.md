## Was bedeutet marktführend?

Im Allgemeinen bedeutet es, dass eine bestimmte Plattform oder ein Betriebssystem:

- auf vielen Geräten läuft
- weit verbreitet ist
- eine große Nutzerbasis hat
- ein starkes Software- und Support-Ökosystem besitzt

Dies kann sich beziehen auf:

- installierte Exemplare
- aktive Nutzung
- Marktanteil
- Verbreitung in Unternehmen

## 1. Windows

Windows ist weiterhin das verbreitetste **Desktop**-Betriebssystem.

Stärken:

- sehr breite Hardwareunterstützung
- viele geschäftliche und alltägliche Softwareanwendungen
- starke Unternehmenspräsenz
- bei Spielen und allgemeiner PC-Nutzung sehr stark

## 2. macOS

macOS ist das Desktop-Betriebssystem von Apple.

Stärken:

- für Apple-Hardware optimiert
- stabile, einheitliche Umgebung
- starke Integration mit anderen Apple-Geräten
- auch in kreativen und Entwicklerumgebungen beliebt

## 3. Linux

Der Desktop-Anteil von Linux ist geringer, aber fachlich sehr wichtig.

Stärken:

- Open Source
- viele verschiedene Distributionen
- flexible Anpassbarkeit
- starke Präsenz bei Entwicklern, Systemadministratoren und im Serverbereich

## 4. Android und iOS

Über alle Plattformen betrachtet sind **Android** und **iOS** sehr bestimmend.

Wichtig jedoch:

- es handelt sich hauptsächlich um **mobile Betriebssysteme**
- der Themenkatalog behandelt mobile Hardware und Android/iOS-Kenntnisse gesondert

## 5. Serverseitige Verbreitung

Bei Servern hat Linux eine herausragende Bedeutung.

Häufige Beispiele:

- Ubuntu Server
- Red Hat Enterprise Linux
- Debian
- SUSE
- Oracle Linux

Daneben ist auch wichtig:

- Windows Server

## Warum ist das in der Prüfung wichtig?

Denn hier reicht es nicht, nur Namen zu nennen, man muss wissen:

- welches System in welcher Umgebung stark ist
- was der Unterschied zwischen Desktop- und Mobilmarkt ist
- warum „marktführend" nicht in jeder Kategorie dasselbe bedeutet

## Prüfungstaugliche Formulierung

> Die marktführenden Betriebssysteme hängen von der Kategorie ab.  
> Im Desktop-Bereich ist Windows am weitesten verbreitet, daneben sind macOS und Linux wichtig.  
> Im mobilen Bereich dominieren Android und iOS.  
> Im Serverumfeld haben Linux und Windows Server große Bedeutung.  
> Die genaue Marktsituation kann sich im Laufe der Zeit ändern, deshalb muss man immer angeben, über welche Plattform und welchen Zeitpunkt man spricht.

## Häufige Prüfungsfehler

- Desktop- und Mobilmarkt verwechseln.
- Behaupten, dass Linux überall marktführend ist.
- Behaupten, dass macOS hardwareseitig genauso universell ist wie Windows.
- Nicht angeben, dass die Marktsituation zeitabhängig ist.
- Android als Desktop-Betriebssystem nennen.

## Schnelle Selbstkontrolle

1. Welches ist das verbreitetste Desktop-Betriebssystem?
2. Welche zwei Systeme dominieren im mobilen Bereich?
3. Welche Betriebssystemfamilie ist bei Servern sehr wichtig?
4. Warum muss man bei Marktanteilen ein Datum oder einen Zeitraum nennen?
5. An welche Hardware ist macOS gebunden?

## Kurzantworten zur Selbstkontrolle

1. **Windows**
2. **Android** und **iOS**
3. Vor allem **Linux**, aber auch **Windows Server** ist wichtig
4. Weil sich die Marktanteile im Laufe der Zeit ändern
5. An **Apple**-Hardware
