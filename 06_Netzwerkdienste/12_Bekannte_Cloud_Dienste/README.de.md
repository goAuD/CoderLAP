# Bekannte Cloud Dienste

## Schneller visueller Überblick

| Dienst | Typ |
|---|---|
| AWS | hauptsächlich IaaS / PaaS |
| Microsoft Azure | hauptsächlich IaaS / PaaS |
| Google Cloud | hauptsächlich IaaS / PaaS |
| Microsoft 365 | hauptsächlich SaaS |
| Google Workspace | hauptsächlich SaaS |
| iCloud | hauptsächlich Endbenutzer-Cloud-Dienst |
| Dropbox | dateibasierter Cloud-Dienst / SaaS-Charakter |

## Warum ist es nützlich, bekannte Beispiele zu kennen?

Weil bei Prüfungen oft nicht nur Definitionen gefragt werden, sondern auch Beispiele.

Es ist gut, wenn man mindestens kennt:

- drei große Infrastrukturanbieter
- zwei bekannte SaaS-Beispiele

## Typische Infrastruktur- und Plattformanbieter

### AWS

Bekannte große Cloud-Plattform.

### Microsoft Azure

In Unternehmens- und Entwicklerumgebungen sehr verbreitet.

### Google Cloud

Bekannt für starkes Angebot bei Daten-, KI- und Plattformdiensten.

## Typische SaaS- und Endbenutzer-Beispiele

### Microsoft 365

Online-Office- und Kollaborationspaket.

### Google Workspace

Cloud-Produktivitätspaket.

### iCloud

An das Apple-Ökosystem gebundener Cloud-Dienst.

### Dropbox

Für Dateifreigabe und Synchronisierung bekannter Cloud-Dienst.

## Worauf muss man achten?

Nicht jeder Cloud-Dienst fällt in dieselbe Kategorie.

Zum Beispiel:

- AWS = hauptsächlich Infrastruktur und Plattform
- Microsoft 365 = fertige Software als Dienst

## Prüfungstaugliche Formulierung

> Bekannte Cloud-Dienste sind zum Beispiel AWS, Microsoft Azure und Google Cloud
> als Infrastruktur- und Plattformanbieter sowie Microsoft 365,
> Google Workspace, iCloud oder Dropbox
> als eher SaaS- oder Endbenutzer-Cloud-Lösungen.  
> Bei der Prüfung ist es wichtig, das Beispiel eines Dienstes dem richtigen Cloud-Modell zuordnen zu können.

## Häufige Prüfungsfehler

- Alle Cloud-Dienste als SaaS bezeichnen.
- Glauben, dass AWS nur Speicherplatz ist.
- Microsoft 365 als Infrastrukturdienst bezeichnen.
- Keine konkreten Beispiele nennen können.

## Schnelle Selbstkontrolle

1. Nenne drei bekannte Infrastruktur- oder Plattform-Cloud-Anbieter.
2. Nenne zwei bekannte SaaS- oder Endbenutzer-Cloud-Dienste.
3. Ist Microsoft 365 eher IaaS oder SaaS?
4. Ist Azure eher Infrastruktur/Plattform oder ein fertiges Office-Paket?
5. Warum ist es wichtig, Beispiele zu kennen?

## Kurzantworten zur Selbstkontrolle

1. AWS, Azure, Google Cloud
2. Microsoft 365, Google Workspace, iCloud, Dropbox
3. Eher SaaS
4. Eher Infrastruktur/Plattform
5. Weil bei Prüfungen oft auch konkrete Anbieternamen gefragt werden
