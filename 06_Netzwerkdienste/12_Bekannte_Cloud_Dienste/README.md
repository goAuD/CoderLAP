# Bekannte Cloud Dienste

## Lényeg 30 másodpercben

A cloud szolgáltatások sokfélék lehetnek.

Vannak:

- infrastruktúra-központú megoldások
- fejlesztői platformok
- végfelhasználói SaaS szolgáltatások

Ismert példák:

- `AWS`
- `Microsoft Azure`
- `Google Cloud`
- `Microsoft 365`
- `Google Workspace`
- `iCloud`
- `Dropbox`

## Gyors vizuális kép

| Szolgáltatás | Típus |
|---|---|
| AWS | főleg IaaS / PaaS |
| Microsoft Azure | főleg IaaS / PaaS |
| Google Cloud | főleg IaaS / PaaS |
| Microsoft 365 | főleg SaaS |
| Google Workspace | főleg SaaS |
| iCloud | főleg végfelhasználói cloud szolgáltatás |
| Dropbox | fájlalapú cloud szolgáltatás / SaaS jelleg |

## Miért hasznos ismert példákat tudni?

Mert vizsgán sokszor nem csak definíciót kérdeznek, hanem példát is.

Jó, ha tudsz legalább:

- három nagy infrastruktúra-szolgáltatót
- két ismert SaaS példát

## Tipikus infrastruktúra- és platformszolgáltatók

### AWS

Ismert nagy cloud platform.

### Microsoft Azure

Vállalati és fejlesztői környezetben nagyon gyakori.

### Google Cloud

Erős adat-, AI- és platformszolgáltatási kínálattal ismert.

## Tipikus SaaS és végfelhasználói példák

### Microsoft 365

Online irodai és együttműködési csomag.

### Google Workspace

Felhős produktivitási csomag.

### iCloud

Apple ökoszisztémához kapcsolódó cloud szolgáltatás.

### Dropbox

Fájlmegosztásra és szinkronizálásra ismert cloud szolgáltatás.

## Mire figyelj?

Nem minden cloud szolgáltatás ugyanabba a kategóriába esik.

Például:

- AWS = főleg infrastruktúra és platform
- Microsoft 365 = kész szoftver szolgáltatásként

## Vizsgán jól használható megfogalmazás

> Ismert cloud szolgáltatások például az AWS, a Microsoft Azure és a Google Cloud mint infrastruktúra- és platformszolgáltatók, valamint a Microsoft 365, a Google Workspace, az iCloud vagy a Dropbox mint inkább SaaS vagy végfelhasználói cloud megoldások.  
> Vizsgán fontos, hogy a szolgáltatás példáját a megfelelő cloud modellhez tudjuk kapcsolni.

## Gyakori vizsgahibák

- Minden cloud szolgáltatást SaaS-nak nevezni.
- Azt hinni, hogy az AWS csak tárhely.
- A Microsoft 365-öt infrastruktúraszolgáltatásnak mondani.
- Nem tudni konkrét példát mondani.

## Gyors önellenőrzés

1. Mondj három ismert infrastruktúra- vagy platform-cloud szolgáltatót.
2. Mondj két ismert SaaS vagy végfelhasználói cloud szolgáltatást.
3. A Microsoft 365 inkább IaaS vagy SaaS jellegű?
4. Az Azure inkább infrastruktúra/platform vagy kész irodai csomag?
5. Miért fontos példákat tudni?

## Rövid válaszok az önellenőrzéshez

1. AWS, Azure, Google Cloud
2. Microsoft 365, Google Workspace, iCloud, Dropbox
3. Inkább SaaS
4. Inkább infrastruktúra/platform
5. Mert vizsgán gyakran kérnek konkrét szolgáltatóneveket is
