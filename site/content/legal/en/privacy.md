# Privacy Policy

## Who is responsible

- Project name: CoderLAP
- Site type: static educational website
- Contact: repository maintainers through the public project repository

## What we collect

- No user accounts
- No sign-up forms
- No comments on the published static site by default
- No analytics tracking by default

## What may still be processed

- Web hosting infrastructure may process IP addresses, timestamps, and request metadata in server or access logs
- Browser requests may include standard technical headers required to serve the page

## Legal basis

- Article 6(1)(f) GDPR for legitimate interests where technical logs are needed to operate and secure the site

## Your rights

- Access your personal data
- Rectify inaccurate data
- Request erasure where applicable
- Restrict processing where applicable
- Object to processing based on legitimate interests
- Request data portability where applicable
