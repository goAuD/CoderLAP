# Web 2.0

## Schneller visueller Überblick

| Web 1.0 Charakter | Web 2.0 Charakter |
|---|---|
| eher statische Seiten | interaktives, partizipatives Web |
| eher Lesen | Lesen + Schreiben + Teilen |
| wenig nutzergenerierter Inhalt | viel User-Generated Content |

## Was ist das Wesentliche an Web 2.0?

Web 2.0 ist die Idee des partizipativen Webs.

Das bedeutet, dass der Benutzer:

- nicht nur Inhalte konsumiert
- sondern sie auch erstellt, bearbeitet und teilt

## Welche Beispiele gehören dazu?

- soziale Netzwerke
- Wikis
- Videoplattformen
- Foren
- Bewertungen und Kommentare

## Warum ist das wichtig?

Weil Web 2.0 große Veränderungen gebracht hat:

- in der Kommunikation
- im Community-Building
- im Marketing
- im Wissensaustausch
- in Online-Geschäftsmodellen

## Welche Vorteile gibt es?

- schneller Informationsaustausch
- gemeinschaftliche Zusammenarbeit
- Nutzerbeteiligung
- einfacheres Publizieren

## Welche Risiken gibt es?

- Fake News
- Datenschutzprobleme
- Online-Mobbing
- Plattformabhängigkeit
- Manipulation und Filterblase

## Nicht verwechseln

| Begriff | Was bedeutet das? |
|---|---|
| Web 2.0 | partizipatives, interaktives Web |
| Internet | das Netzwerk selbst, nicht dasselbe wie das Web |
| Social Media | ein Teil von Web 2.0, nicht der gesamte Begriff |

## Prüfungstaugliche Formulierung

> Web 2.0 bezeichnet jene Phase der Web-Entwicklung,
> in der Benutzer Inhalte nicht nur konsumieren,
> sondern auch selbst erstellen und teilen.  
> Dazu gehören beispielsweise Blogs, Wikis, soziale Netzwerke
> und andere Plattformen,
> die auf nutzergeneriertem Inhalt basieren.  
> Der Vorteil liegt in der Zusammenarbeit und schnellen Kommunikation,
> das Risiko kann jedoch Datenschutzprobleme, Fake News und Manipulation sein.

## Häufige Prüfungsfehler

- Web 2.0 nur als "neues Webdesign" zu bezeichnen.
- User-Generated Content nicht zu erwähnen.
- Social Media mit dem gesamten Web 2.0 gleichzusetzen.
- Zu vergessen, dass es sich auch um einen historischen Begriff handelt.

1. Was ist das Wesentliche an Web 2.0?
2. Nenne drei Web 2.0 Beispiele.
3. Was ist der Unterschied zwischen dem Charakter von Web 1.0 und Web 2.0?
4. Nenne einen Vorteil.
5. Nenne ein Risiko.

## Kurzantworten zur Selbstkontrolle

1. Der Benutzer erstellt aktiv auch Inhalte
2. Blog, Wiki, soziales Netzwerk
3. Web 2.0 ist interaktiver und partizipativer
4. Schnellerer Wissensaustausch
5. Fake News oder Datenschutzproblem
