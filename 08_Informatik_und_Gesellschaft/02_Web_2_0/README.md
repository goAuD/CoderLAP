# Web 2.0

## Gyors vizuális kép

| Web 1.0 jelleg | Web 2.0 jelleg |
|---|---|
| inkább statikus oldalak | interaktív, részvételi web |
| inkább olvasás | olvasás + írás + megosztás |
| kevés felhasználói tartalom | sok user-generated content |

## Mi a Web 2.0 lényege?

A Web 2.0 a részvételi web gondolata.

Ez azt jelenti, hogy a felhasználó:

- nem csak fogyasztja a tartalmat
- hanem létrehozza, módosítja, megosztja is

## Milyen példák tartoznak ide?

- közösségi hálózatok
- wikik
- videómegosztók
- fórumok
- értékelések és kommentek

## Miért fontos?

Mert a Web 2.0 nagy változást hozott:

- kommunikációban
- közösségépítésben
- marketingben
- tudásmegosztásban
- online üzleti modellekben

## Milyen előnyei vannak?

- gyors információmegosztás
- közösségi együttműködés
- felhasználói részvétel
- könnyebb publikálás

## Milyen kockázatai vannak?

- álhírek
- adatvédelmi problémák
- online zaklatás
- platformfüggőség
- manipuláció és buborékhatás

## Ne keverd össze

| Fogalom | Mit jelent? |
|---|---|
| Web 2.0 | részvételi, interaktív web |
| internet | maga a hálózat, nem ugyanaz mint a web |
| social media | a Web 2.0 egyik része, nem a teljes fogalom |

## Vizsgán jól használható megfogalmazás

> A Web 2.0 a web fejlődésének azt a szakaszát jelöli, amelyben a felhasználók már nemcsak fogyasztják, hanem létre is hozzák és megosztják a tartalmat.  
> Ide tartoznak például a blogok, wikik, közösségi oldalak és más felhasználói tartalomra épülő platformok.  
> Előnye az együttműködés és a gyors kommunikáció, kockázata viszont az adatvédelmi probléma, az álhír és a manipuláció lehet.

## Gyakori vizsgahibák

- A Web 2.0-t pusztán "új webdesignnak" nevezni.
- Nem megemlíteni a user-generated contentet.
- A social mediát a teljes Web 2.0-val azonosítani.
- Elfelejteni, hogy ez történeti fogalom is.
