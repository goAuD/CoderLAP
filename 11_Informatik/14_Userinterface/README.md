# Userinterface

## Lényeg 30 másodpercben

A **user interface** (`UI`, felhasználói felület) az a felület, amelyen keresztül a felhasználó kapcsolatba lép egy rendszerrel.

Ez lehet:

- képernyős felület
- gomb, menü, űrlap
- érintőképernyős vezérlés
- hangalapú vagy más logikai interfész

Röviden:

- a UI az ember és a rendszer találkozási pontja

## Gyors vizuális kép

| UI-elem | Feladata |
|---|---|
| gomb | művelet indítása |
| menü | navigáció |
| input mező | adatbevitel |
| ikon | vizuális jelentés |
| visszajelzés | állapot közlése |

## Mi az a user interface?

A felhasználói felület az a fizikai vagy logikai eszközrendszer, amellyel a felhasználó használni tud egy alkalmazást, eszközt vagy folyamatot.

Webes környezetben ez tipikusan:

- oldalszerkezet
- navigáció
- űrlapok
- gombok
- táblázatok
- állapotjelzések
- hibavisszajelzések

## Miért fontos?

Mert a felhasználó nem a forráskódot látja, hanem a felületet.

Egy jó UI:

- érthető
- következetes
- könnyen kezelhető
- gyorsan megtanulható
- egyértelmű visszajelzést ad

Egy rossz UI:

- zavaró
- lassítja a munkát
- növeli a hibázás esélyét
- csökkenti az elégedettséget

## A jó UI alapelvei

- egyértelműség
- konzisztencia
- átlátható navigáció
- jó olvashatóság
- látható visszajelzés
- hozzáférhetőség

## UI és UX: ne keverd össze

| Fogalom | Mit jelent? |
|---|---|
| UI | maga a felület és annak elemei |
| UX | a teljes felhasználói élmény |

A UI inkább a "mit lát és mivel kattint", a UX inkább a "milyen érzés és mennyire hatékony" oldal.

## Mire kell figyelni fejlesztőként?

- A felület legyen érthető első ránézésre.
- A hibákhoz legyen világos hibaüzenet.
- Az űrlapok legyenek logikusak.
- A mobil és desktop nézet eltéréseit tervezni kell.
- Az akadálymentesség nem opcionális extra, hanem minőségi követelmény.

## Vizsgán jól használható megfogalmazás

> A user interface a rendszer azon része, amelyen keresztül a felhasználó kapcsolatba lép az alkalmazással vagy eszközzel.  
> Ide tartoznak például a gombok, menük, űrlapok, ikonok és visszajelző elemek.  
> Egy jó felhasználói felület érthető, következetes, könnyen kezelhető és támogatja a hibamentes használatot.

## Gyakori vizsgahibák

- A UI-t összekeverni a UX-szel.
- Csak "kinézetként" beszélni róla.
- Nem megemlíteni a visszajelzés szerepét.
- Nem hozni rá konkrét elempéldát.
