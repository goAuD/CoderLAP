# Soll-Ist-Analyse

## Schneller visueller Überblick

| Element | Bedeutung |
|---|---|
| Ist | aktuelle Situation |
| Soll | gewünschte Situation |
| Gap / Abweichung | was gelöst werden muss |
| Maßnahme | wie wir zum Ziel gelangen |

## Was ist eine Soll-Ist-Analyse?

Es ist eine Analysemethode, die vergleicht:

- den aktuellen Prozess oder das aktuelle System
- den gewünschten zukünftigen Zustand

Dann wird gesucht:

- was fehlt
- was schlecht funktioniert
- was verbessert oder eingeführt werden muss

## Wozu ist sie gut?

- hilft, das Problem zu klären
- unterstützt die Planung
- bildet eine Grundlage für Entwicklungsprioritäten
- macht die nötigen Schritte sichtbar

## Typische Beispiele

- aktuelles System ist zu langsam, Ziel ist ein schnellerer Betrieb
- aktueller Prozess ist manuell, Ziel ist die Automatisierung
- aktuelle Software unterstützt eine bestimmte Funktion nicht, Ziel ist deren Einführung

## Soll-Ist-Analyse und Fehlersuche: nicht verwechseln

| Begriff | Fokus |
|---|---|
| Fehlersuche | ein konkretes Problem finden |
| Soll-Ist-Analyse | Vergleich von aktuellem und Zielzustand |

## Prüfungstaugliche Formulierung

> Bei der Soll-Ist-Analyse vergleichen wir den aktuellen Zustand mit dem angestrebten Zielzustand.  
> Ziel der Analyse ist das Erkennen von Abweichungen, die Identifikation von Lücken und die Bestimmung der notwendigen Maßnahmen.  
> Sie kann eine wichtige Grundlage für die Projektplanung und Entwicklungsentscheidungen sein.

## Häufige Prüfungsfehler

- Die Begriffe `Ist` und `Soll` vertauschen.
- Die Abweichungsanalyse nicht erwähnen.
- Die Methode auf einen einfachen Fehlerbericht reduzieren.
- Auslassen, dass sie auch eine Planungsgrundlage sein kann.

## Schnelle Selbstkontrolle

1. Was bedeutet `Ist`?
2. Was bedeutet `Soll`?
3. Warum ist diese Analyse nützlich?
4. Was ist eines der Hauptergebnisse der Analyse?
5. Worin unterscheidet sie sich von einer Fehlersuche?

## Kurzantworten zur Selbstkontrolle

1. Den aktuellen Zustand
2. Den gewünschten Zielzustand
3. Weil sie Entwicklungsbedarf sichtbar macht
4. Die Liste der Abweichungen und Maßnahmen
5. Sie sucht nicht nur Fehler, sondern vergleicht den Zielzustand
