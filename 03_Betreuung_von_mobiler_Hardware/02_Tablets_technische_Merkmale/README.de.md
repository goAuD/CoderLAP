# Tablets technische Merkmale

## Schneller visueller Überblick

| Merkmal               | Warum ist es beim Tablet wichtig?  |
| --------------------- | ---------------------------------- |
| größeres Display      | Lesen, Video, Arbeit, Zeichnen     |
| Multitasking          | parallele Nutzung mehrerer Apps    |
| Stylus-Unterstützung  | Notizen, Zeichnen, Bildung         |
| Tastaturunterstützung | Produktivität                      |
| lange Akkulaufzeit    | mobiler Einsatz                    |
| optionales Cellular   | mobiles Internet mit SIM oder eSIM |

## Was ist ein Tablet?

Ein Tablet kann als Kategorie zwischen Smartphone und Laptop betrachtet werden.

Seine Stärken:

- größeres Display als beim Telefon
- trotzdem noch tragbar
- gut für Medienkonsum, Lernen, Präsentationen und in vielen Fällen auch zum Arbeiten

## Wichtige technische Merkmale

## 1. Display

Beim Tablet besonders wichtig:

- Displaygröße
- Auflösung
- Helligkeit
- Farbqualität
- Bildwiederholrate

Warum?

- wird für längeres Lesen genutzt
- wird für Videos genutzt
- mehr Inhalt ist gleichzeitig sichtbar
- beim Multitasking ist die größere Fläche nützlich

## 2. Leistung

Auch im Tablet gibt es:

- SoC / Prozessor
- GPU
- RAM

Das ist wichtig bei:

- Mehrfensterbetrieb
- kreativen Anwendungen
- Notizen
- Video- oder Grafikarbeiten

## 3. RAM und Speicher

Genauso wichtig wie beim Smartphone:

- RAM = Verwaltung laufender Anwendungen
- Speicher = Dateien, Apps, Medien

Häufige Tablet-Nutzung:

- PDFs
- Notizen
- Videos
- Präsentationen
- Zeichnungen

## 4. Akku

Beim Tablet ist die Akkulaufzeit besonders wichtig, da es häufig:

- unterwegs genutzt wird
- im Bildungsbereich eingesetzt wird
- in Meetings verwendet wird
- für längeren Medienkonsum genutzt wird

## 5. Konnektivität

Ein Tablet kann haben:

- `Wi-Fi`
- `Bluetooth`
- optional `4G` oder `5G`
- `eSIM` oder physische SIM
- `USB-C`

Das bestimmt zum Beispiel:

- Online-Nutzung
- Anschluss von Zubehör
- Dateiübertragung

## 6. Zubehörunterstützung

Dies ist eines der Schlüsselthemen beim Tablet.

Häufiges Zubehör:

- Stylus / digitaler Stift
- Tastatur
- Maus oder Trackpad

Das macht das Tablet geeignet für:

- Notizen
- Zeichnen
- Dokumentenbearbeitung
- Präsentationen

## 7. Multitasking und Nutzung des großen Displays

Einer der wichtigen Vorteile des Tablets gegenüber dem Smartphone:

- komfortablere Darstellung mehrerer Inhalte gleichzeitig
- besseres Split-Screen- oder Mehrfensterarbeiten

## 8. Kamera und Audio

Obwohl das Tablet kein primäres Fotogerät ist, kann es dennoch wichtig sein für:

- Videoanrufe
- Dokumenten-Fotografie
- Bildungs- oder Geschäftszwecke

Beim Audio kann wichtig sein:

- mehrere Lautsprecher
- Mikrofonqualität

## Tablet vs Smartphone

| Tablet                                     | Smartphone                                     |
| ------------------------------------------ | ---------------------------------------------- |
| größeres Display                           | kleiner, passt in die Tasche                   |
| besser zum Lesen und Multitasking          | besser für einhändige, schnelle mobile Nutzung |
| wird häufig mit Stift / Tastatur verwendet | wird seltener auf diese Weise genutzt          |
| weniger „telefonzentriert"                 | ist auch primär ein Kommunikationsgerät        |

## Prüfungstaugliche Formulierung

> Ein Tablet ist ein tragbares, berührungsgesteuertes Gerät mit einem größeren Display als ein Smartphone.  
> Wichtige technische Merkmale sind die Displaygröße und -qualität,
> der Prozessor, der RAM, der Speicher, der Akku,
> die drahtlosen Verbindungen und die Multitasking-Fähigkeit.  
> Tablets unterstützen häufig auch einen digitalen Stift und eine Tastatur
> und sind daher auch für Bildungs- und Produktivitätsaufgaben geeignet.

## Häufige Prüfungsfehler

- Das Tablet einfach als „großes Telefon" bezeichnen.
- Die Rolle von Stylus und Tastatur nicht erwähnen.
- Den Multitasking-Vorteil weglassen.
- Glauben, dass ein Tablet nur für Medienkonsum gut ist.
- Keinen Unterschied zwischen Smartphone und Tablet nennen können.
