# Smartphones technische Merkmale

## Schneller visueller Überblick

| Merkmal | Was wird betrachtet? |
|---|---|
| Display | Größe, Auflösung, Technologie, Helligkeit, Bildwiederholrate |
| Leistung | SoC, CPU, GPU, NPU |
| Arbeitsspeicher | RAM |
| Speicher | interner Speicher, Erweiterbarkeit |
| Energie | Akku, Laden |
| Kamera | Hauptkamera, Ultra Wide, Zoom, Video |
| Konnektivität | 4G/5G, Wi-Fi, Bluetooth, NFC, GPS |
| Sicherheit | PIN, Fingerabdruck, Gesichtserkennung |
| Software | Android oder iOS, Updates |

## Was ist ein Smartphone?

Ein Smartphone ist ein fortschrittliches Mobiltelefon, das über die klassischen Telefonfunktionen hinaus:

- Apps ausführt
- im Web surft
- eine Kamera verwendet
- Standortbestimmung durchführt
- biometrische Identifikation handhabt
- auch digitale Zahlungen ermöglichen kann

## Wichtige technische Merkmale

## 1. Display

Beim Display werden typischerweise folgende Punkte betrachtet:

- Größe, z. B. `6.1"` oder `6.7"`
- Auflösung
- Pixeldichte
- Displaytechnologie, z. B. `OLED`
- Helligkeit
- Bildwiederholrate, z. B. `60 Hz` oder `120 Hz`

Warum ist das wichtig?

- Lesbarkeit
- Bildqualität
- Flüssigkeit
- Energieverbrauch

## 2. Prozessor / SoC

Einer der wichtigsten Bestandteile des Smartphones ist das **SoC** (`System on a Chip`).

Dieses kann enthalten:

- CPU
- GPU
- AI / NPU-Einheiten
- Kommunikationssteuerung
- Sicherheitselemente

Es bestimmt unter anderem:

- die Geschwindigkeit
- die Energieeffizienz
- die Leistungsfähigkeit der Kamera- und AI-Funktionen

## 3. Arbeitsspeicher und Speicher

### RAM

Der RAM ist der temporäre Speicher für laufende Anwendungen und Prozesse.

Mehr RAM hilft in der Regel bei:

- dem Offenhalten mehrerer Apps
- einem flüssigeren Multitasking

### Interner Speicher

Dieser speichert:

- das System
- die Anwendungen
- die Fotos
- die Videos
- die Dokumente

Typische Größen:

- `128 GB`
- `256 GB`
- `512 GB`

## 4. Akku und Laden

Wichtige Merkmale:

- Akkukapazität
- kabelgebundenes Laden
- kabelloses Laden
- Energieeffizienz

Praktischer Aspekt:

- nicht nur die mAh zählen, sondern auch die Optimierung des Systems

## 5. Kamera

In einem modernen Smartphone können mehrere Kameras vorhanden sein:

- Hauptkamera
- Ultraweitwinkelkamera
- Telekamera
- Frontkamera

Wichtige Merkmale:

- Sensorgröße
- Auflösung
- Stabilisierung
- Zoom
- Videofähigkeiten
- Nachtmodus

## 6. Konnektivität

Typische Mobilverbindungen:

- `4G`
- `5G`
- `Wi-Fi`
- `Bluetooth`
- `NFC`
- `GPS` / Standortbestimmung
- `eSIM` oder physische SIM

Diese bestimmen:

- Internetzugang
- Geräteverbindung
- Zahlung
- Navigation

## 7. Sensoren

Häufige Sensoren:

- Beschleunigungsmesser
- Gyroskop
- Näherungssensor
- Lichtsensor
- Kompass
- biometrische Sensoren

Verwendung:

- Bildschirmdrehung
- automatische Helligkeit
- Spiele
- Navigation
- Fitnessfunktionen

## 8. Sicherheit und Identifikation

Beispiele:

- PIN
- Passcode
- Fingerabdruckleser
- Gesichtserkennung

Heute ist dies bereits ein grundlegender Bestandteil der mobilen Nutzung.

## 9. Betriebssystem

Der Betrieb des Smartphones wird von einem mobilen Betriebssystem gesteuert.

Die wichtigsten Beispiele:

- **Android**
- **iOS**

Das OS beeinflusst:

- die Bedienung
- die Verfügbarkeit von Anwendungen
- die Updates
- das Sicherheitsmodell

## Worauf ist bei der Prüfung zu achten?

Konkrete Modelle müssen nicht auswendig gelernt werden, aber man muss verstehen:

- was welches technische Merkmal bedeutet
- was die Nutzbarkeit beeinflusst
- welches Bauteil wofür zuständig ist

## Prüfungstaugliche Formulierung

> Ein Smartphone ist ein fortschrittliches Mobiltelefon, das für Kommunikation, Internetzugang und das Ausführen von Anwendungen geeignet ist.  
> Wichtige technische Merkmale sind das Display, der Prozessor bzw. SoC, der RAM, der Speicher, der Akku, die Kameras, die drahtlosen Verbindungen, die Sensoren und die biometrische Identifikation.  
> Der Betrieb wird von einem mobilen Betriebssystem wie Android oder iOS gesteuert.

## Häufige Prüfungsfehler

- Nur Kamera und Display erwähnen.
- Die Begriffe RAM und Speicher verwechseln.
- Nicht wissen, welche Rolle das SoC spielt.
- Glauben, dass allein die mAh alles über die Akkulaufzeit aussagen.
- Konnektivität und Sensoren nicht erwähnen.
